# RelayStream Sia Storage Proof Demo

This folder documents the Sia Storage proof-of-work demonstration created for the RelayStream Sia grant proposal review.

## Purpose

The demonstration was created to show practical experience using Sia Storage and to demonstrate technical understanding of the Sia network.

The video includes:

- Sia dashboard overview
- Active storage contracts
- Bucket and object management
- File upload workflow
- Storage organization
- Review of the Sia Developer Portal
- Evaluation of Sia as part of RelayStream's decentralized media storage architecture

## Relationship to RelayStream

RelayStream is evaluating Sia as a decentralized storage layer for media ingestion, archival storage, metadata coordination, and storage orchestration workflows.

The current demo does not represent a finished Sia-native ingestion engine. It represents hands-on testing and evaluation of Sia Storage workflows that inform the design of the proposed RelayStream Open Ingestion Engine.
